# Todo List
Basic TODO app with Node and AngularJS V1

1/ Install the project dependencies : <b>npm install</b>

2/ Start mongod : <b>mongod</b>

3/ Start node : <b>node server.js</b>

4/ Browse the app : <b>http://localhost:8080</b>

# Development

Start Gulp : <b>gulp</b>
